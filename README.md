Использование генеративных моделей ИИ в проекте

Используемая модель:
Claude Sonnet 4.6 (Anthropic)
Адрес: https://claude.ai
Инструмент: Claude Code (CLI-версия)

Описание применения:
1. Scroll-анимации на главной странице (index.html)
Цель: создать анимации, управляемые скроллом страницы, для нескольких секций.

2. Scroll-анимация секции «Мы ценим» (about.html, `a_blok3`)
Цель: при скролле напротив каждого слова (Свободу, Любовь к музыке, Наслаждение моментами) появляется блок с изображением, при переходе к следующему слову предыдущий блок исчезает.

3. Анимация видео/изображения на странице «О проекте» (about.html, `a_blok1`)
Цель: видео начинает большим, по скроллу уменьшается до натурального размера и встаёт под текст, после чего скролл продолжается к следующим секциям.

4. Интерактивная доска мероприятий (events.html)
Цель: карточки-афиши расположены на разных слоях, при движении мыши сдвигаются с разной скоростью (эффект глубины).

5. Страница артистов (artists.html) — scroll с видео
Цель: при скролле по очерёдности подсвечивается имя артиста (увеличивается, становится белым), рядом появляется панель с видео и описанием; предыдущая панель закрывается.

6. Страница оплаты (payment.html)
Цель: создать разделение цифр по 4 в форме ввода данных карты и добавить отображение выбранного тарифа.

7. Адаптивная вёрстка под мобильные устройства
Цель: адаптировать сложные анимации проекта под экраны до 450px.

8. Preloader (экран предварительной загрузки)
Цель: показать экран загрузки до полной загрузки контента.
